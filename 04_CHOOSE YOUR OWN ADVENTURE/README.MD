# Project 04 — Choose Your Own Adventure

## What it does
A text-based adventure game where the player makes choices at each fork
in the road — go left or right, swim or walk, cross a bridge or turn back.
Every decision leads to a different outcome. Only one path leads to winning.

## What I learned
- How to use nested `if/elif/else` statements to build branching logic
- How to chain multiple decisions together into one story flow
- How to use `.lower()` to handle capitalised user input
- How to personalise output using a variable stored from `input()`

## What was hard
- Keeping track of which `if` block belongs to which branch —
  indentation errors were easy to make here
- Making sure every possible input had a response so the
  program never silently fails

## What I would add next
- A map that shows the player where they are in the story
- More branching paths so there are multiple ways to win
- A replay option at the end without restarting the program

# Project 01 — Quiz Game

## What it does

A terminal-based computer science quiz that asks 5 questions and tracks your score in real time.

## What I learned

- How to use `if/else` statements to check answers
- How to use `.lower()` and `.capitalize()` to handle user input cleanly
- How to track and calculate a score dynamically
- How to use f-strings to display results

## What was hard

- Remembering to normalise user input so capitalisation does not break the answer check
- Spotting a bug where `q4` checks `q1` instead of `q4` — a copy-paste error I identified after writing the code

## What I would add next

- A list of questions so the quiz is easy to expand without repeating code
- A timer for each question
- A leaderboard that saves scores to a file
